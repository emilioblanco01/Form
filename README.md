# Formulario de Registro (Vite / React)


Este es un template pare el formulario de registro de un usuario.

Esta desarrollado con Vite / React.

## Requisitos Previos

* Node V16.18.1
* Yarn V 1.22.19

## Instalacion Previa.

Una vez clonado el Repositorio, realizar los siguientes pasos:

1.- yarn
2.- yarn dev
